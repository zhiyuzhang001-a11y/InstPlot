## 📚 主要功能

### 1️⃣ 数据导入

**支持的文件格式**：
- ✅ CSV (`.csv`)
- ✅ 文本文件 (`.txt`)
- ✅ Excel (`.xlsx`, `.xls`)

**导入方式**：
- 点击工具栏 **"打开文件"** 按钮
- 直接**拖拽文件**到软件窗口

### 2️⃣ 数据可视化

**轴选择**：
- 通过下拉菜单选择 X 轴和 Y 轴数据列
- 支持中文列名和 LaTeX 格式

**图表类型**：
- 折线图、散点图
- 自动图例生成
- 网格线显示

### 3️⃣ 数据处理功能

#### 🔄 对称处理
将数据沿 Y 轴镜像对称，适用于对称性分析

#### 📏 归一化
将数据缩放到 [0, 1] 范围，便于不同量级数据对比

#### 🧹 去背底
移除数据的基线背景，突出主要特征

### 4️⃣ 交互式操作

**鼠标操作**：
- **左键拖拽** - 平移图表
- **滚轮** - 缩放图表
- **矩形选择** - 精确查看局部数据
- **右键** - 重置视图

**键盘快捷键**：
- `Ctrl + O` - 打开文件
- `Ctrl + S` - 保存图片
- `Ctrl + Z` - 撤回操作
- `Ctrl + E` - 导出数据

### 5️⃣ 数据导出

**图片导出**：
- PNG（推荐用于网页）
- JPG（压缩格式）
- SVG（矢量图，可无损缩放）
- PDF（适合论文发表）

**数据导出**：
- 导出处理后的数据为 CSV 格式
- 保留所有计算结果
